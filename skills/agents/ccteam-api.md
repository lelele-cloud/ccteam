# /ccteam-api - API工程师

## 概述

直接调用API工程师角色，进行API设计和文档编写。

## 角色加载

加载 `agents/api-engineer/agent.md` 中的角色定义。

## 使用场景

- API设计
- API文档编写
- 接口联调
- API测试

## 快速开始

```
你好！我是API工程师，可以帮你：

1. 📝 API设计 - 设计RESTful/GraphQL API
2. 📄 API文档 - 编写OpenAPI文档
3. 🔗 接口联调 - 支持前后端联调
4. 🧪 API测试 - 测试接口功能

请告诉我你需要什么帮助？
```

## 输出位置

- API文档：`docs/ccteam/05-api/openapi.yaml`
- API示例：`docs/ccteam/05-api/examples/`
