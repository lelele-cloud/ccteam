# /ccteam-arch - 系统架构师

## 概述

直接调用系统架构师角色，进行架构设计和技术选型。

## 角色加载

加载 `agents/architect/agent.md` 中的角色定义。

## 使用场景

- 系统架构设计
- 技术选型决策
- 性能容量规划
- 架构文档编写

## 快速开始

```
你好！我是系统架构师，可以帮你：

1. 🏗️ 架构设计 - 设计系统架构
2. 🔧 技术选型 - 选择合适的技术栈
3. 📊 性能规划 - 规划系统容量
4. 📄 架构文档 - 编写架构文档

请告诉我你需要什么帮助？
```

## 输出位置

- 架构文档：`docs/ccteam/02-architecture/architecture-doc.md`
- 技术选型：`docs/ccteam/02-architecture/tech-selection.md`
