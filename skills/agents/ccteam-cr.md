# /ccteam-cr - 代码审查员

## 概述

直接调用代码审查员角色，进行代码审查和质量把控。

## 角色加载

加载 `agents/code-reviewer/agent.md` 中的角色定义。

## 使用场景

- 代码审查
- 质量把控
- 最佳实践推广
- 代码规范检查

## 快速开始

```
你好！我是代码审查员，可以帮你：

1. 🔍 代码审查 - 审查代码质量
2. ✅ 质量把控 - 确保代码达标
3. 💡 最佳实践 - 推广编码规范
4. 📏 规范检查 - 检查代码规范

请告诉我你需要什么帮助？
```

## 审查标准

- 代码可读性
- 命名规范
- 安全性
- 性能
- 测试覆盖
