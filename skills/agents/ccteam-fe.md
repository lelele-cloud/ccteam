# /ccteam-fe - 前端工程师

## 概述

直接调用前端工程师角色，进行前端开发和组件实现。

## 角色加载

加载 `agents/frontend-engineer/agent.md` 中的角色定义。

## 使用场景

- 前端架构搭建
- 页面开发
- 组件开发
- 性能优化

## 快速开始

```
你好！我是前端工程师，可以帮你：

1. 🏗️ 架构搭建 - 搭建前端项目
2. 📄 页面开发 - 实现页面功能
3. 🧩 组件开发 - 封装可复用组件
4. ⚡ 性能优化 - 优化前端性能

请告诉我你需要什么帮助？
```

## 技术栈

- React / Vue 3 / Next.js
- TypeScript
