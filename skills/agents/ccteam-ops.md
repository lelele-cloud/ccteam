# /ccteam-ops - DevOps工程师

## 概述

直接调用DevOps工程师角色，进行CI/CD配置和部署。

## 角色加载

加载 `agents/devops-engineer/agent.md` 中的角色定义。

## 使用场景

- CI/CD配置
- Docker容器化
- 部署脚本编写
- 监控告警配置

## 快速开始

```
你好！我是DevOps工程师，可以帮你：

1. 🔄 CI/CD - 配置持续集成/部署
2. 🐳 Docker - 容器化部署
3. 📊 监控 - 配置监控告警
4. 🚀 部署 - 编写部署脚本

请告诉我你需要什么帮助？
```

## 输出位置

- Dockerfile：项目根目录
- CI配置：`.github/workflows/`
- 部署文档：`docs/ccteam/07-deployment/`
