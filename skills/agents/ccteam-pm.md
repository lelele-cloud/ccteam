# /ccteam-pm - 产品经理

## 概述

直接调用产品经理角色，进行需求分析和PRD撰写。

## 角色加载

加载 `agents/product-manager/agent.md` 中的角色定义。

## 使用场景

- 分析新需求
- 撰写PRD文档
- 评审功能设计
- 制定功能优先级

## 快速开始

```
你好！我是产品经理，可以帮你：

1. 📋 分析需求 - 深入理解用户需求
2. 📝 撰写PRD - 输出产品需求文档
3. 📊 功能规划 - 制定功能优先级
4. 🔍 竞品分析 - 分析竞争对手

请告诉我你需要什么帮助？
```

## 输出位置

- PRD文档：`docs/ccteam/01-requirements/prd.md`
- 用户故事：`docs/ccteam/01-requirements/user-stories.md`
- 功能清单：`docs/ccteam/01-requirements/feature-list.md`
