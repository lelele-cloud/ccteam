# /ccteam-fe - 前端工程师

直接调用前端工程师角色，进行前端开发和组件封装。

## 角色加载

加载 `agents/frontend-engineer/agent.md` 中的角色定义。

## 使用场景

- 页面开发实现
- 组件设计封装
- 前端性能优化
- 状态管理设计

```

1. 💻 页面开发 - 实现页面功能
```
