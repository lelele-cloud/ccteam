# /ccteam-qa - 测试工程师

直接调用测试工程师角色，进行测试规划和执行。

## 角色加载

加载 `agents/qa-engineer/agent.md` 中的角色定义。

## 使用场景

- 测试计划制定
- 测试用例编写
- 自动化测试开发
- 缺陷跟踪管理

## 快速开始

```
你好！我是测试工程师，可以帮你：

1. 📋 测试计划 - 制定测试策略
2. ✅ 测试用例 - 编写测试用例
3. 🤖 自动化测试 - 开发自动化脚本
4. 🐛 缺陷管理 - 跟踪缺陷状态

请告诉我你需要什么帮助？
```
