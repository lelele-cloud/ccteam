# CCTeam - 恢复项目

## 概述

恢复中断的项目，从上次停止的地方继续。

## 使用方法

当用户调用 `/ccteam-resume` 时：

### 1. 检查项目状态

读取 `docs/ccteam/project-status.json`。

如果文件不存在：
```
未找到进行中的项目。

请使用以下命令开始新项目：
- /ccteam - 开始新项目向导
- /ccteam-new - 直接开始新项目
- /ccteam-feature - 添加新功能
```

### 2. 展示项目信息

```
🔄 恢复项目

项目名称：{{projectName}}
上次更新：{{updatedAt}}

当前阶段：{{currentStage.name}}
阶段状态：{{currentStage.status}}
负责角色：{{currentStage.agent}}

上次产出：
{{lastOutputs}}
```

### 3. 确认恢复

```
是否从当前阶段继续？

1. ✅ 继续当前阶段
2. 🔙 回退到上一阶段
3. ⏭️ 跳过当前阶段
4. 📋 查看详细状态
```

### 4. 恢复执行

根据用户选择：
- 继续：加载对应角色，继续工作
- 回退：更新状态，重新开始上一阶段
- 跳过：标记跳过，进入下一阶段
- 查看：调用 `/ccteam-status`

### 5. 加载上下文

恢复时自动加载：
- 项目基本信息
- 已完成阶段的产出物
- 当前阶段的进度和产出
- 未解决的问题
