# 组件规范

## 按钮组件

### 类型
- 主按钮: 用于主要操作
- 次按钮: 用于次要操作
- 文字按钮: 用于轻量操作
- 图标按钮: 用于空间受限场景

### 尺寸
- 大: 40px高度
- 中: 32px高度（默认）
- 小: 24px高度

### 状态
- 默认 / 悬停 / 按下 / 禁用 / 加载中

## 输入框组件

### 类型
- 单行输入
- 多行输入
- 密码输入
- 搜索输入

### 状态
- 默认 / 聚焦 / 错误 / 禁用 / 只读

## 表单组件

- 选择器（单选/多选）
- 日期选择器
- 时间选择器
- 开关
- 复选框
- 单选框

## 导航组件

- 顶部导航
- 侧边导航
- 面包屑
- 标签页
- 步骤条

## 数据展示

- 表格
- 卡片
- 列表
- 空状态
- 加载状态

## 反馈组件

- 消息提示
- 通知
- 对话框
- 气泡确认框
- 进度条

## 组件命名规范

格式: `模块_类型_名称_状态`

示例:
- btn_primary_default
- btn_primary_hover
- input_text_focus
- icon_close_24
