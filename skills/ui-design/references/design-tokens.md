# 设计令牌(Design Tokens)

## 颜色系统

### 品牌色
- 主色: #1890FF (用于主要操作、链接)
- 辅助色: #52C41A (用于成功状态)

### 功能色
- 成功: #52C41A
- 警告: #FAAD14
- 错误: #FF4D4F
- 信息: #1890FF

### 中性色
- 标题: #262626
- 正文: #595959
- 次要: #8C8C8C
- 禁用: #BFBFBF
- 边框: #D9D9D9
- 分割: #F0F0F0
- 背景: #F5F5F5

## 字体系统

### 字体族
- 中文: PingFang SC, Microsoft YaHei
- 英文: -apple-system, Helvetica Neue
- 代码: SFMono-Regular, Consolas

### 字号层级
| 用途 | 字号 | 行高 | 字重 |
|------|------|------|------|
| H1标题 | 30px | 38px | 600 |
| H2标题 | 24px | 32px | 600 |
| H3标题 | 20px | 28px | 600 |
| H4标题 | 16px | 24px | 600 |
| 正文 | 14px | 22px | 400 |
| 辅助文字 | 12px | 20px | 400 |

## 间距系统

基础单位: 4px

| 名称 | 值 | 用途 |
|------|-----|------|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 紧凑元素间距 |
| md | 16px | 标准元素间距 |
| lg | 24px | 分组间距 |
| xl | 32px | 区块间距 |
| xxl | 48px | 大区块间距 |

## 圆角

| 名称 | 值 | 用途 |
|------|-----|------|
| sm | 2px | 小按钮、标签 |
| md | 4px | 默认圆角 |
| lg | 8px | 卡片、弹窗 |
| full | 50% | 圆形头像 |

## 阴影

| 名称 | 值 | 用途 |
|------|-----|------|
| sm | 0 1px 2px rgba(0,0,0,0.1) | 按钮hover |
| md | 0 4px 12px rgba(0,0,0,0.15) | 卡片、下拉 |
| lg | 0 8px 24px rgba(0,0,0,0.2) | 弹窗 |
