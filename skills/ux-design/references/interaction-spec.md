# 交互规范

## 反馈原则

| 响应时间 | 处理方式 |
|----------|----------|
| <100ms | 无需反馈 |
| 100ms-1s | 按钮状态变化 |
| 1s-10s | 加载指示器 |
| >10s | 进度条+预估时间 |

## 表单交互

1. **验证时机**
   - 实时验证: 输入时即时反馈
   - 失焦验证: 离开输入框时验证
   - 提交验证: 提交表单时统一验证

2. **错误提示**
   - 就近显示: 在错误字段附近
   - 明确说明: 告诉用户如何修正
   - 保留输入: 不清空用户已输入内容

3. **表单规范**
   - 必填项标识清晰(*)
   - 支持键盘导航(Tab)
   - Enter键提交表单

## 导航交互

- 当前位置有明确标识
- 返回操作一键可达
- 层级面包屑导航
- 支持快捷键

## 状态反馈

### 加载状态
- 骨架屏(首次加载)
- 加载指示器(刷新)
- 下拉刷新动画

### 空状态
- 清晰的说明文字
- 引导用户操作的按钮
- 友好的插图

### 错误状态
- 错误描述清晰
- 提供解决方案
- 允许重试操作

## 动效规范

| 类型 | 时长 | 用途 |
|------|------|------|
| 微交互 | 100-200ms | 按钮点击、开关 |
| 页面切换 | 200-300ms | 路由变化 |
| 弹窗动画 | 200-400ms | 模态框显隐 |
| 复杂动画 | 300-500ms | 页面过渡 |
